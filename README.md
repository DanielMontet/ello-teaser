## Getting Started

Clone this repository:

```
git clone https://github.com/DanielMontet/ello-teaser.git
```

#### then

```
cd ello-teaser
```

#### then install dependancies

```bash
npm install
```

#### or

```
yarn install
```

#### RUN

```
npm run start
```

#### or

```
yarn start
```
